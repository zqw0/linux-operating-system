# 内核同步  
## 临界区和竞争条件  
**临界区**：就是访问和操作共享数据的代码段。比如：两个线程同时修改a变量，就有可能造成不可预知的变量赋值。  
**竞争条件**：就是说两个线程有可能同时修改a变量，同时竞争a。这就是竞争条件。  
**同步**：为了避免上述这种竞争条件，我们称之为同步。  
eg:
```
i++的例子。
线程1                   线程2
获得i(7)               获得i(7)
增加i(7->8)        -----
------                     增加i(7->8)
写回i(8)               -----
------                      写回i(8)
```
